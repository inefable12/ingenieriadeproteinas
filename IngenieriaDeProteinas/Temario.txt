Base de datos de proteínas

1. UniProt: Una base de datos completa que cataloga secuencias de proteínas y anotaciones funcionales, vinculando los datos estructurales con las funciones biológicas.
2. Proteín Databank (PDB): El principal archivo global de datos estructurales 3D de macromoléculas biológicas. Proporciona información detallada sobre las estructuras de las proteínas, las interacciones de ligandos y las condiciones experimentales.
3. AlphaFold: https://alphafold.ebi.ac.uk/
